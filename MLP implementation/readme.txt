Label para os dados presentes no arquivo 'inputDataset.csv':
A,B,C,D,Area,Peso_Fora,Densidade_Arquimedes

Significado das colunas:
A - Maior comprimento do ovo
B - Altura do ovo
C - Distancia entre a ponta inferior e a linha de comprimento
D - Distancia entre a ponta superior e a linha de comprimento
Area - Area do ovo (PDI)
Peso_Fora - Peso do ovo
Densidade_Arquimedes - D.A do ovo

Label para os dados presentes no arquivo 'Output.csv':
Embriodiagnostico

Significado da coluna:
O embriodiagnóstico define o status do ovo após
o período de incubação. Nascido ou Morto

Observação:
1 - Nascido
0 - Morto
